# SMART-FINANCE-MANAGER-PERSONAL-EXPENSE-INTELLIGENCE-SYSTEM
Smart Finance Manager &amp; Personal Expense Intelligence System (SFM-PEIS) is a Python-based app for managing personal finances. It tracks transactions, categorizes expenses, manages budgets and subscriptions, and analyzes spending using NumPy, Pandas, and Matplotlib to provide insights and detect unusual activity.
